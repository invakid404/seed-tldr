# hakyll-init

> Generate a new Hakyll sample blog.
> More information: <https://github.com/jaspervdj/hakyll-init>.

- Generate a new Hakyll sample blog:

`hakyll-init {{path/to/directory}}`

- Show help for `hakyll-init`:

`hakyll-init --help`
