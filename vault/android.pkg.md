# pkg

> Package management utility for Termux.
> More information: <https://wiki.termux.com/wiki/Package_Management>.

- Upgrade all installed packages:

`pkg upgrade`

- Install a package:

`pkg install {{package}}`

- Uninstall a package:

`pkg uninstall {{package}}`

- Reinstall a package:

`pkg reinstall {{package}}`

- Search for a package:

`pkg search {{package}}`
