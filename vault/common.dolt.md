# dolt

> Dolt is a SQL database that you can fork, clone, branch, merge, push and pull just like a Git repository.
> More information: <https://github.com/dolthub/dolt>.

- Execute a dolt subcommand:

`dolt {{subcommand}}`

- List available subcommands:

`dolt help`
