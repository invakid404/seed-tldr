# xed

> Opens files for editing in Xcode.

- Open file in Xcode:

`xed {{file1}}`

- Open file(s) in Xcode, create if it doesn't exist:

`xed -c {{filename1}}`

- Open a file in Xcode and jump to line number 75:

`xed -l 75 {{filename}}`
