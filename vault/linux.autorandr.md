# autorandr

> Automatically change screen layout.

- Save the current screen layout:

`autorandr -s {{profile_name}}`

- Show the saved profiles:

`autorandr`

- Change the profile:

`autorandr -l {{profile_name}}`

- Set the default profile:

`autorandr -d {{profile_name}}`
