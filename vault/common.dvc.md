# dvc

> Data Version Control: like `git` for data.
> More information: <https://dvc.org/>.

- Check the DVC version:

`dvc --version`

- Display general help:

`dvc --help`

- Display help about a specific subcommand:

`dvc {{subcommand}} --help`

- Execute a DVC subcommand:

`dvc {{subcommand}}`
