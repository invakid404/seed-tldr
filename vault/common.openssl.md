# openssl

> OpenSSL cryptographic toolkit.
> More information: <https://www.openssl.org>.

- Print a list of available subcommands:

`openssl help`

- Print options for a specific subcommand:

`openssl help {{x509}}`

- Print the version of OpenSSL:

`openssl version`
