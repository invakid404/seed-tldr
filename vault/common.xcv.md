# xcv

> Cut, copy, and paste in the command-line.
> More information: <https://github.com/busterc/xcv>.

- Cut a file:

`xcv x {{input_file}}`

- Copy a file:

`xcv c {{input_file}}`

- Paste a file:

`xcv v {{output_file}}`

- List files available for pasting:

`xcv l`
