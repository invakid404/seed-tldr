# free

> Display amount of free and used memory in the system.

- Display system memory:

`free`

- Display memory in Bytes/KB/MB/GB:

`free -{{b|k|m|g}}`

- Display memory in human readable units:

`free -h`

- Refresh the output every 2 seconds:

`free -s {{2}}`
