# cargo help

> Display help on `cargo` and its subcommands.
> More information: <https://doc.rust-lang.org/cargo/commands/cargo-help.html>.

- Display general help:

`cargo help`

- Display help for a subcommand:

`cargo help {{subcommand}}`
