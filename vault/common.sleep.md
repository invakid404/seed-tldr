# sleep

> Delay for a specified amount of time.
> More information: <https://pubs.opengroup.org/onlinepubs/9699919799/utilities/sleep.html>.

- Delay in seconds:

`sleep {{seconds}}`

- Execute a specific command after 20 seconds delay:

`sleep 20 && {{command}}`
