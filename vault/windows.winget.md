# winget

> Windows Package Manager CLI.
> More information: <https://docs.microsoft.com/windows/package-manager/winget>.

- Install a package:

`winget install {{package}}`

- Display information about a package:

`winget show {{package}}`

- Search for a package:

`winget search {{package}}`
