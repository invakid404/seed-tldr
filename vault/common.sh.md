# sh

> Bourne shell.
> The standard command language interpreter.

- Start interactive shell:

`sh`

- Execute a command:

`sh -c {{command}}`

- Run commands from a file:

`sh {{file.sh}}`

- Run commands from stdin:

`sh -s`
