# wormhole

> Get things from one computer to another, safely.
> More information: <https://magic-wormhole.readthedocs.io/en/latest/>.

- Send a file:

`wormhole send {{path/to/file}}`

- Receive a file:

`wormhole receive {{wormhole_code}}`

- Send raw text:

`wormhole send`
