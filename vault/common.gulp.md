# gulp

> JavaScript task runner and streaming build system.
> Tasks are defined within gulpfile.js at the project root.
> More information: <https://github.com/gulpjs/gulp-cli>.

- Run the default task:

`gulp`

- Run individual tasks:

`gulp {{task}} {{othertask}}`
