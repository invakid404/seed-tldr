# cmatrix

> Shows a scrolling Matrix like screen in the terminal.
> More information: <https://github.com/abishekvashok/cmatrix>.

- Enable asynchronous scrolling:

`cmatrix -a`

- Display red text:

`cmatrix -C {{red}}`

- Enable rainbow mode:

`cmatrix -r`

- Set screen update delay to 2 centiseconds (20 milliseconds):

`cmatrix -u {{2}}`
