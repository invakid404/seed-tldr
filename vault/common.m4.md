# m4

> Macro processor.
> More information: <https://www.gnu.org/software/m4>.

- Process macros in a file:

`m4 {{path/to/file}}`

- Define a macro before processing files:

`m4 -D{{macro_name}}={{macro_value}} {{path/to/file}}`
