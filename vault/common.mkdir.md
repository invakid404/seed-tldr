# mkdir

> Creates a directory.
> More information: <https://www.gnu.org/software/coreutils/mkdir>.

- Create a directory in current directory or given path:

`mkdir {{directory}}`

- Create directories recursively (useful for creating nested dirs):

`mkdir -p {{path/to/directory}}`
