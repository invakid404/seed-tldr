# sbcl

> High performance Common Lisp compiler.
> More information: <http://www.sbcl.org/>.

- Start an SBCL interactive shell (REPL):

`sbcl`

- Execute a Lisp script:

`sbcl --script {{path/to/script.lisp}}`
