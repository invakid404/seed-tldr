# stun

> Classic STUN client.
> More information: <https://manned.org/stun.1>.

- Make a STUN request:

`stun {{stun.1und1.de}}`

- Make a STUN request and specify the source port:

`stun {{stun.1und1.de}} -p {{4302}}`
