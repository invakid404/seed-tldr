# zotero

> Manage your bibliographies.
> More information: <https://www.zotero.org/support>.

- Run with the GUI:

`zotero`

- Run in headless mode:

`zotero --headless`

- Run with a specific profile:

`zotero -P {{profile}}`

- Run the Migration Assistant:

`zotero --migration`
