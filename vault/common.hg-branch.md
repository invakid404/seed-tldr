# hg branch

> Create or show a branch name.
> More information: <https://www.mercurial-scm.org/doc/hg.1.html#branch>.

- Show the name of the currently active branch:

`hg branch`

- Create a new branch for the next commit:

`hg branch {{branch_name}}`
