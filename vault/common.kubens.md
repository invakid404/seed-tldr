# kubens

> Utility to switch between Kubernetes namespaces.
> More information: <https://github.com/ahmetb/kubectx>.

- List the namespaces:

`kubens`

- Change the active namespace:

`kubens {{name}}`

- Switch to the previous namespace:

`kubens -`
