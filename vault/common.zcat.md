# zcat

> Print data from gzip compressed files.

- Print the uncompressed contents of a gzipped file to the standard output:

`zcat {{file.txt.gz}}`

- Print compression details of a gzipped file to the standard output:

`zcat -l {{file.txt.gz}}`
